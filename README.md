
SESAME is a scrypt as a proof-of-work algorithm coin.
 - 2.2 minute block targets
 - subsidy halves in 500k blocks (~3 years)
 - ~60 million total coins
 - 100 coins per block
 - 4000 blocks to retarget difficulty
